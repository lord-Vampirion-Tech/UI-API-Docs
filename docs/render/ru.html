render.lua

Файл отвечает за низкоуровневую отрисовку в терминал.

Примечание: При обычном использовании этот объект не будет затронут вами.
Это скорее справочная информация, если вы собираетесь писать свои модули.

Что это?

render.lua хранит внутренний буфер экрана и предоставляет методы для вывода пикселей в терминал с использованием вспомогательных функций.

Основные методы

Функция Что делает
render:init(x, y, w, h) Создаёт буфер нужного размера и задаёт его положение.
render:clear(bg) Очищает буфер и заполняет фон цветом bg. (по стандарту bg = "f")
render:pixel(x, y, ch, fg, bg, clip) Рисует один символ в указанной точке.
render:blit(x, y, text, fg, bg, clip, mode) Выводит строку текста с цветами. Параметр mode ориентация текста.
render:rect(x, y, w, h, bg, clip) Рисует закрашенный прямоугольник.
render:rectOutline(x, y, w, h, fg, bg, clip) Рисует рамку вокруг области.
render:present() Переносит содержимое буфера на терминал.

Примеры работы функций

render:init(x, y, w, h)

Создаеть экземпляр render

  • x, y — координаты верхнего левого угла буфера;
  • w, h — ширина и высота буфера;
  • clip — область в которой он может рисовать. (используеться внутри модулей);
  • Обычно используют для создания рабочего области под интерфейс, но можно использовать и вне ui.lua.

local render = require("render")
local r = render(1, 1, 20, 10)

render:clear(bg)

Очищает весь буфер и заполняет его выбранным цветом.

  • bg — цвет фона.

local render = require("render")
local r = render(1, 1, 20, 10)

r:clear("f")

Функции рисования !

render:pixel(x, y, ch, fg, bg, clip)

Рисует символ в нужной позиции нужным цветом.

  • x, y — позиция символа;
  • ch — символ для рисования. По стандарту равен " ";
  • fg — цвет текста/символа;
  • bg — цвет фона;
  • clip — необязательное ограничение области.

local render = require("render")
local r = render(1, 1, 20, 10)

r:pixel(3, 2, "#", "e", "0")

render:blit(x, y, text, fg, bg, clip, mode)

Работает как обычный term.blit(). Отрисовывает сразу строчку символов.

  • x, y — координаты начала строки;
  • text — строка для вывода;
  • fg — цвет текста;
  • bg — цвет фона;
  • clip — ограничение области, если нужно;
  • mode — режим вывода, например "h" для скрытого/специального режима.

local render = require("render")
local r = render(1, 1, 20, 10)

r:blit(2, 2, "Hello", "0", "7")

render:rect(x, y, w, h, bg, clip)

Рисует прямоугольник в определенной позцици, нужного размера и цвета.

  • x, y — позиция верхнего левого угла;
  • w, h — ширина и высота прямоугольника;
  • bg — цвет заливки;
  • clip — необязательное ограничение области.

local render = require("render")
local r = render(1, 1, 20, 10)

r:rect(2, 2, 6, 3, "7")

render:rectOutline(x, y, w, h, fg, bg, clip)

Рисует рамку прямоугольника в определенной позцици, нужного размера и цвета.

  • x, y — позиция рамки;
  • w, h — размеры рамки;
  • fg — цвет линий рамки;
  • bg — цвет фона внутри рамки;
  • clip — необязательное ограничение области.

local render = require("render")
local r = render(1, 1, 20, 10)

r:rectOutline(2, 2, 6, 3, "e", "0")

render:present()

Выводит данные из буфера на экран терминала/монитора.

  • Не принимает аргументов;
  • Отправляет текущий буфер на экран и делает результат видимым.
  • Обычно вызывается в конце кадра или после всех изменений.

local render = require("render")
local r = render(1, 1, 20, 10)

r:clear("f")
r:blit(2, 2, "OK", "0", "7")
r:present()
