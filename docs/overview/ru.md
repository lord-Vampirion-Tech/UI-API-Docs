# Обзор библиотеки UI !

[ссылка тест](#overview/fast_start)


[ссылка тест](#ui)

Это небольшая библиотека для создания простых интерфейсов в CC-Tweaked. Она построена вокруг буфера отрисовки, общего контейнера UI и набора компонентов.

## Основная идея

Библиотека работает по событийной модели:

1. Создаётся экземпляр UI-контейнера.
2. В него добавляются элементы. (панели, кнопки, и тд)
3. UI обрабатывает события мыши и перерисовывает интерфейс.

## Основные части

| Функция | Что делает                                                                                  |
| ------- | ------------------------------------------------------------------------------------------- |
| render  | буфер отрисовки, который хранит символы, цвета текста и фон.                                |
| utils   | служебные функции: создание классов, выравнивание текста, работа с координатами и clipping. |
| ui      | корневой контейнер приложения, который хранит элементы и запускает цикл событий.            |
| modules | готовые компоненты интерфейса. По желаюнию можете дописывать и свои модули                  |

Вся библиотке построена по принципу OOP и модульности. Каждый модуль может быть как обычным классом, так и наследованным от другова, получая его свойства и функции.

## Быстрый старт {fast_start}

> Примечание: 
> обьяснение как это работает описанно далее в [ui](#ui)

```lua
local ui = require("ui")

local title = ui.label(2, 2, 20, "Hello", "c")
local button = ui.button(2, 4, 12, 3, "OK", "c", function()
  title.text = "Good Job!"
end)

ui:add(title)
ui:add(button)
ui:start()
```

## Что можно сделать

- создавать простые окна и панели;
- добавлять кнопки и переключатели;
- отображать списки и выпадающие меню;
- использовать вкладки и слайдеры;
- управлять положением, видимостью и событиями элементов.
