# ui.lua

Файл является корневым контейнером библиотеки.  

> Примечание:
> При обычном использовании большинство методов не будет затронут вами.  
> Это скорее справочная информация, как работает код ui.lua

## Что это ?

ui.lua собирает все доступные модули из папки `modules`, добавляет их в объект `ui` и предоставляет методы для добавления элементов, рисования и обработки событий.

---

## Основные методы

| Функция                       | Что делает                                                                                                          |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `ui:add(element)`             | Добавляет элемент в список UI и привязывает его к родителю.                                                         |
| `ui:draw()`                   | Отрисовывает/Перерисовывает все видимые элементы.                                                                   |
| `ui.handleEvent(event, clip)` | Обходит элементы в обратном порядке и вызывает их обработку событий.                                                |
| `ui:start()`                  | Запускает бесконечный цикл событий: ждёт события, обрабатывает их, перерисовывает интерфейс и выводит его на экран. |

## Пример

```lua
local ui = require("ui")

local title = ui.label(2, 2, 20, "Hello")
ui:add(title)
ui:start()
```

> Предупреждение:
> пока что код ui не умеет адеквтано (по моему мнению) работать с логикой основной программы.  
> При желании можно запускать паралельные циклы `ui:start` и вашей основной программы.
