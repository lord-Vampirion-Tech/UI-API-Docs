# utils.lua

Файл содержит общие вспомогательные функции для всей библиотеки.

> Примечание:
> При обычном использовании этот объект не будет затронут вами.  
> Это скорее справочная информация, если вы собираетесь писать свои модули.

## Что это?

utils.lua реализует базовые механики: создание классов, выравнивание текста, вычисление глобальных координат, работу с clipping и пересечением областей.

---

## Основные функции

| Функция                                | Что делает                                                                       |
| -------------------------------------- | -------------------------------------------------------------------------------- |
| `utils.createClass(className, parent)` | Создаёт класс с поддержкой наследования и вызова `init` при создании экземпляра. |
| `utils.align(text, width, mode)`       | Обрезает или выравнивает текст по ширине.                                        |
| `utils.getGlobalPos(self, gx, gy)`     | Возвращает абсолютные координаты элемента с учётом его родителей.                |
| `utils.getClip(self)`                  | Возвращает clip-область элемента.                                                |
| `utils.inClip(x, y, clip)`             | Проверяет, находится ли точка внутри clip.                                       |
| `utils.intersectClip(a, b)`            | Возвращает пересечение двух clip-областей.                                       |

---
 
## Примеры работы функций

### `utils.createClass(className, parent)`

Функция создания нового класса.

- `className` — наименование нового класса. Обязательно;
- `parent` — от какого класса наследоваться. Не обязательно.

```lua
local utils = require("utils")
local newClass = utils.createClass("newClass")

-- с наследованием
local base = require("modules.base")
local newClass = utils.createClass("newClass",base) 
-- или передать его напрямую как параметр `("newClass",require("modules.base")) `
```

---

### `utils.align(text, width, mode)`

Возвращает выровненный текст в заданном режиме и при заданной длине.

- `text` — текст, который нужно выровнять;
- `width` — ширина строки, по которой нужно выровнять;
- `mode` — режим выравнивания. По умолчанию выбран режим `"l"`, то есть — слева;
  - режимы работы:
  - `"l"` — выровнять по левому краю;
  - `"c"` — выровнять по центру;
  - `"r"` — выровнять по правому краю.

```lua
local utils = require("utils")

local text = "test text"

local newText_l = utils.align(text,20) -- для "l" можно не писать
-- выдаст строку "test text           "
local newText_c = utils.align(text,20,"c")
-- выдаст строку "     test text      "
local newText_r = utils.align(text,20,"r")
-- выдаст строку "           test text"
```

### `utils.getGlobalPos(self, gx, gy)`

Возвращает глобальную позицию объекта, исходя из локальной позиции через `self` и учёта позиции родителя. Если родителя нет, просто возвращает позицию.

- `self` — текущий класс, в котором идёт работа;
- `gx, gy` — позиция родителя, передаваемая при вызове функции `:draw()`.
  
```lua
local utils = require("utils")

--
-- допустим мы создали новый класс и пишем функцию для его отрисовки
--

function newClass:draw(render, gx, gy, clip)
-- поля render, gx, gy и clip передаются ранее от родителя, который вызвал эту функцию

-- так мы получим позицию данного объекта
local x, y = utils.getGlobalPos(self, gx, gy)

--
-- а далее мы можем их использовать для чего угодно
--

end
```

### `utils.getClip(self)`

Получить зону, в которой может быть отрисован объект.

- `self` — текущий класс, в котором идёт работа.

```lua
local utils = require("utils")

--
-- допустим мы создали новый класс и пишем функцию для его отрисовки
-- тут нам еще понадобиться получить собственную clip-область если мы делаем контейнер или его подобие
--

local myClip = utils.getClip(self)

-- вот так выглядит обьект `clip`
clip = {
  x, y,
  w, h
}

end
```

---

### `utils.inClip(x, y, clip)`

Проверяет, находится ли позиция внутри clip-области. Возвращает `true` или `false`.

- `x`, `y` — позиция;
- `clip` — clip-область.

```lua
local utils = require("utils")

local inClip = utils.inClip(10, 10, clip)

end
```

> Подсказка:
> По большей части метод используется в объекте render для ограничения рисования.

---

### `utils.intersectClip(a, b)`

Возвращает новую clip-область, полученную из двух входных.

- `a`, `b` — две clip-зоны. Первая — родительская, вторая — собственная.
- Порядок имеет значение.

```lua
local utils = require("utils")

-- создаём 2 clip-зоны

local clip_1 = {
  x = 1, y = 1,
  w = 10, h = 10
}

local clip_2 = {
  x = 5, y = 5,
  w = 10, h = 10
}

-- и вызовем метод intersectClip
local newClip = utils.intersectClip(clip_1, clip_2)

-- он будет равен:
-- x = 5, y = 5
-- w = 5, h = 5
-- так как первый clip обрезал второй

-- если поменять местами параметры в функции, то зона будет равна clip_1

```
