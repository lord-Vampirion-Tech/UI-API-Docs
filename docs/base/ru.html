base.lua

Базовый класс для всех визуальных элементов.

Button

Кнопка используется для обработки событий. Она имеет состояния:

  • normal
  • hover
  • pressed

Основные методы

Функция Что делает
base:init(x, y, w, h) Инициализирует базовые поля элемента.
base:draw(render, gx, gy, clip) Базовый метод отрисовки. В текущей реализации он пустой и предназначен для переопределения.
base:hitTest(x, y, clip) Проверяет, находится ли точка внутри элемента.
base:handleEvent(event, clip) Проверяет попадание мыши в область и вызывает onEvent.
base:onEvent(event) Базовая обработка события. По умолчанию возвращает true.

Что это?

base.lua определяет общие свойства такие как

  • позиция,
  • размеры,
  • видимость,
  • цвет текста и фона,
  • а также базовую логику обработки событий.

Он используеться для создания других классов.

base:init(x, y, w, h)

  • w, h — ширина и высота буфера;
